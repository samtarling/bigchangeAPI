# bigchangeAPI
An PHP library for interfacing with the [BigChange](https://www.bigchange.com/) API


## What works
Not much yet! :sweat_smile:
 - BigChangeAPI\Auth - creates an object with reusable basic auth token
 - BigChangeAPI\Contact - creates a BigChange contact object and loads with contact data
